<br>
## • 사용 언어 및 프레임워크
• HTML <br>
• CSS <br>

• JavaScript <br>
## • 주요 기능
• 회원가입 및 로그인 구현<br>
• 수업 일정 캘린더<br>
• 운동 영상 스트리밍<br>
• 회원 수업 일정 변경 일부 구현<br>
• 공지 사항 기능<br>
## • 보완해야할 점
• 월별 캘린더 보완<br>
• 월별 캘린더에 회원 및 강사 일정 DB 연동<br>
• 데일리 스케쥴 템플릿 구현<br>
• 월별 캘린더와 데일리 스케쥴간의 연동<br>
• 회원 및 강사 스케쥴 변경 구현<br>
• 회원 및 강사 로그인 시 자신의 일정 및 운동영상만 나오도록 연동<br>
• 회원 운동 영상 목록 페이지 보완<br>
• 회원 운동 영상 상세 페이지에 댓글 및 대댓글 기능 구현<br>
